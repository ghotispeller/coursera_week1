Lecture 12 - Anatomy of a CSS Rule

The collection of CSS declarations is called a Stylesheet.

selector {
property: value;
}

Examples:
p {
color: blue;
font-size: 20px;
width: 200px;
}

h1 {
color: green;
font-size: 36px;
text-align: center;
}
