Lecture 03 - Anatomy of an html tag

Tag Structure

Most html tags have an opening and closing tag and they surround content.

  • An example of a tag would be "<p>".
  • <p> content </p>
  • Technically speaking p is the element and <p> is the tag. But the two terms are used for both.
  • No space is allowed right after the opening bracket. Space is allowed anywhere else.
  • Some tags only have an opening tag, such as:
    • <br> (line break at the end of a line) and
    • <hr> (horizontal rule such as the one you can see below this list)

Lecture 04 - Basic HTML Document Structure

Tag attributes

Example: <tr id="tableHeadersId">

Every html element can have predefined attributes. Attributes are a pair of attribute name and attribute value. For example id attributes need to be unique. No other element can have the same id!

  • There must be a space after the element of the opening tag and its attributes.
  • Attributes can only be assigned in the opening tag.

Value of attributes. It's not technically required to surround the attribute with double or single quotes, but you need to watch out for situations like these: "alert('hi')". It doesn't matter which quote you start with, but the inner quotes must be closed first.

Lecture 05 - HTML Content Models

Elements in prior HTML versions

The content model determines which elements are allowed to be nested inside with other element. Before HTML5 there were only block elements or inline elements. Now there are seven models.

Block level elements (similar to HTML5 category "Flow Content")

  1. begin on a new line by default
  2. may contain inline or other block-level elements

Inline elements (similar to HTML5 category "Phrasing Content")

  1. render on the same line by default
  2. may only contain other inline elements

Div and Span elements

Div(block) and Span(inline) are the most generic elements. Div elements can contain Span elements, but not the other way around.

You can have: <div> <span> </span> </div> but not <span> <div> </div> </span>

W3.org lists the seven types of element here: Link.

Lecture 07 - Lists

Lists and Tables are not the same

See the document "unordered lists before html"

  • ul ... unordered list (bullet points)
  • ol ... ordered list (numbered)
  • li ... list item
  • You can indent list items like this:
    • <li> This tag opens the list that contains a sub-list
      • <ul> This tag opens the indented list.
        • <li> "Indented list item 1"
        • </li> "Indented list item 2"
      • </ul> This tag closes the indented list.
    • </li> This tag closes the part that contains an indented list.

Self-closing tags (outdated)

In previous versions of html, esp. xhtml it was possible to use a shorthand notation to create a self-closing tag. A self-closing tag was a tag that will not contain any content, such as <p/> and was useful if you require a placeholder for content that you were planning to insert dynamically at a later point. However, in html5 this is not valid anymore.
